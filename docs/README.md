    1. M0: Course Introduction(2분), [M1: Introduction to the AWS Cloud(20분)](./blob/master/docs/module1/1%20모듈%20소개.md)
    2. [M2: AWS Core Services(45)](./blob/master/docs/module2/1%20모듈%20소개.md)

3. 11월 시험 응시
